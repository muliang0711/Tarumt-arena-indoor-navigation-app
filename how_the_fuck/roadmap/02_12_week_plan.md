# 12 Week Plan

## Weeks 1-2

Learn:

- tilemaps
- tile size
- grids
- world coordinates
- pixel art basics

Build:

- one tiny room with 3 tile types

## Weeks 3-4

Learn:

- palette control
- shading
- wall and corner logic

Build:

- one mini tileset

## Weeks 5-6

Learn:

- Tiled
- atlas layout
- map layers

Build:

- one handcrafted mini floor

## Weeks 7-8

Learn:

- canvas tile rendering
- sprite sheet slicing
- camera control

Build:

- a viewer that renders the handcrafted map

## Weeks 9-10

Learn:

- graph integration
- interactive markers
- route highlighting

Build:

- a hybrid map with pretty visuals and graph logic

## Weeks 11-12

Learn:

- polish
- variation
- shadows
- performance

Build:

- one final showcase map

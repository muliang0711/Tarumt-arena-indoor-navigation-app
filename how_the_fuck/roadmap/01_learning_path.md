# Learning Path

## Practical roadmap

### Phase 1: Fix fundamentals

Study:

- pixel art basics
- tileset rules
- readability

Deliverable:

- one tiny handcrafted map image

### Phase 2: Build a real tile renderer

Upgrade your app so it can:

- load a tile atlas
- map tile IDs to atlas coordinates
- render multiple layers
- place decorations

Deliverable:

- one JSON tilemap rendered from a sprite sheet

### Phase 3: Connect graph data to authored map data

Use graph data for:

- navigation logic
- room metadata
- route calculation
- facility placement rules

Use authored tilemap data for:

- floor visuals
- wall visuals
- decorations
- shadows
- polish

Deliverable:

- a navigation-aware map that still looks handcrafted

### Phase 4: Production quality

Add:

- better labels
- more tile variants
- shadow pass
- object layering
- import/export workflow

Deliverable:

- a polished map viewer instead of a prototype

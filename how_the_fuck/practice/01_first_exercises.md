# First Exercises

## Practice order

Do not start with a giant map.

Start with this sequence:

1. make one `16x16` floor tile
2. make 4 floor variations
3. make wall top, side, and corner tiles
4. make one door tile
5. make 3 simple props
6. build one tiny room
7. build one corridor
8. build one mini floor

## Why this order works

- it teaches consistency
- it keeps the scope small
- it forces reuse
- it helps you notice style problems early

## Best immediate next move

For your exact repo:

1. keep the current graph-based prototype
2. draw a tiny tileset in Aseprite
3. add sprite-sheet rendering to the demo
4. build one handcrafted showcase floor
5. only after that, improve automation

# Checklist And Mistakes

## Common beginner mistakes

Avoid these:

- using too many colors
- mixing tile sizes
- adding detail everywhere
- ignoring contrast
- drawing without a tile system
- relying only on code generation for art
- using blurry scaling
- designing giant maps too early

## Quality checklist

Before calling a map good, check:

- can I read the walkable path instantly?
- do important rooms stand out?
- are wall edges clear?
- is the palette controlled?
- are shadows consistent?
- do repeated tiles have variation?
- does zoom keep pixels crisp?
- is the map understandable without labels?
- do labels improve clarity instead of adding clutter?

## Final rule

Do not chase perfection first.

Aim in this order:

1. readable
2. consistent
3. reusable
4. polished

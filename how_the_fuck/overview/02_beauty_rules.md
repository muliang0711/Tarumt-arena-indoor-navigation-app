# What Makes A Pixel Map Beautiful

## Main visual rules

### Clear shape language

- rooms are easy to read
- paths are obvious
- landmarks stand out

### Good value contrast

- walls are darker
- floors are middle value
- highlights are brighter

### Controlled palette

- not too many colors
- consistent material colors
- limited bright accents

### Tile variation

- floor variants
- corners
- edge tiles
- transitions
- damaged or alternate tiles

### Layering

- ground layer
- wall layer
- object layer
- shadow layer
- overlay layer

### Pixel consistency

- same tile size
- same light direction
- same outline logic
- crisp scaling

## The short version

If a map is readable, consistent, and layered, it already feels much better.

If you add tile variation, good contrast, and controlled detail, it starts to feel polished.

# Core Knowledge

## Stage 1: Absolute beginner

Learn:

- what a tilemap is
- what a tileset is
- what a sprite sheet is
- tile sizes like `16x16`, `24x24`, `32x32`
- layers
- grid coordinates
- collision tiles vs visual tiles
- world space vs screen space

Goal:

- build a tiny map and render it without blur

## Stage 2: Pixel art fundamentals

Learn:

- silhouette
- palette control
- hue shifting
- dithering
- selective outlining
- anti-aliasing in pixel art
- light direction
- shadow placement

Goal:

- draw a small tile set by hand and keep it visually consistent

## Stage 3: Map composition

Learn:

- focal points
- flow
- negative space
- room readability
- landmark placement
- environmental storytelling

Goal:

- make one small room feel clear and interesting

## Stage 4: Tile system design

Learn:

- autotiling basics
- corners
- inside corners
- transitions between materials
- modular tile kits
- atlas organization

Goal:

- create reusable floor and wall tiles instead of redrawing everything

## Stage 5: Rendering and tooling

Learn:

- canvas rendering basics
- draw order
- hit testing
- camera and zoom
- culling
- tile indexing
- data formats

Goal:

- render a tilemap from a sprite sheet cleanly

## Stage 6: Advanced polish

Learn:

- decoration passes
- lighting passes
- animated tiles
- overlay effects
- performance tuning
- authoring workflow

Goal:

- turn a functional map into a polished map

# Tools And Skills

## Best beginner tool stack

- **Aseprite** for pixel art and tilesets
- **Tiled** for map editing
- **React + Canvas** if you want to keep your current frontend approach
- **JSON** for map data during early development

## Art skills that matter most

If you only focus on three art topics first, choose:

1. value control
2. tile variation
3. composition

## Technical skills that matter most

If you only focus on three technical topics first, choose:

1. tilemap data structures
2. sprite sheet rendering
3. layered scene composition

## Subjects to learn over time

- basic drawing
- pixel art fundamentals
- color theory
- light and shadow
- composition
- tileset design
- level design
- map readability
- sprite sheets
- tilemaps
- canvas rendering
- camera systems
- coordinate transforms
- JSON modeling
- tool pipeline design
- performance optimization

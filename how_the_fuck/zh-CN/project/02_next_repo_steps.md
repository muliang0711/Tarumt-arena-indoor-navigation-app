# 这个仓库下一步该做什么

## 最高价值的改进项

### 1. 用 tile atlas 渲染替代大量纯矩形绘制

现在 `pixelRenderer.ts` 里还有很多 `fillRect`。

下一步：

- 做一张 tileset PNG
- 定义 tile ID
- 用 `drawImage` 渲染 tile

### 2. 把逻辑 tile 和美术 tile 分开

不要让一套 tile 系统同时承担所有事情。

拆成：

- 逻辑图结构 tile
- 视觉美术 tile

### 3. 加图层

目标结构：

- 地面层
- 墙层
- 物件层
- 覆盖层
- 交互层

### 4. 给重点空间加手工设计

不要把所有重要区域都完全自动生成。

建议手工制作：

- 大厅
- 特殊房间
- 地标
- 装饰角落

### 5. 从单一配色升级到主题包

以后你应该支持类似这些主题：

- 医院
- 办公室
- 商场
- 学校
- 工厂

每个主题可以定义：

- 配色
- 标签样式
- 高亮颜色
- 可选的 tile atlas

# 地图类型

## “2D 地图”其实有两种完全不同的东西

### 1. 像素艺术 tilemap

这是常见的游戏风格地图，由 tile 组成：

- 地板
- 墙
- 草地
- 水面
- 道具
- 阴影
- 装饰

这种地图通常会在 Aseprite、Tiled 之类的工具里手工制作。

### 2. 数据驱动的图结构地图

这就是你当前 `demo` 项目正在做的事情：

- 节点会变成房间或设施
- 边会变成走廊
- JSON 数据会被转换成可读布局

这种方式适合：

- 室内导航
- 建筑平面布局
- 路线可视化
- 楼层预览

## 重要区别

一个漂亮的像素艺术地图，本质上主要依赖美术和 tile 工作流。

一个图结构地图，本质上主要依赖数据和渲染工作流。

你的项目已经有了不错的图结构流程，现在缺的是更强的视觉 tilemap 流程。

# 工具与技能

## 推荐的新手工具组合

- **Aseprite** 用来画像素图和 tileset
- **Tiled** 用来编辑地图
- **React + Canvas** 如果你想继续沿用现在的前端方案
- **JSON** 用来承载早期地图数据

## 最重要的美术技能

如果一开始只能先专注三个主题，就优先学：

1. 明暗控制
2. tile 变化设计
3. 构图

# 核心知识

## 阶段 1：绝对入门

学习：

- 什么是 tilemap
- 什么是 tileset
- 什么是 sprite sheet
- `16x16`、`24x24`、`32x32` 这类 tile 尺寸
- 图层
- 网格坐标
- 碰撞 tile 和视觉 tile 的区别
- world space 和 screen space 的区别

目标：

- 做一个很小的地图，并且无模糊地渲染出来

## 阶段 2：像素艺术基础

学习：

- 轮廓
- 配色控制
- 色相偏移
- 抖动
- 选择性描边
- 像素艺术里的抗锯齿
- 光照方向
- 阴影摆放

目标：

- 手工画出一套小 tileset，并保持风格一致

## 阶段 3：地图构图

学习：

- 焦点
- 动线
- 留白
- 房间可读性
- 地标摆放
- 环境叙事

目标：

- 让一个小房间既清楚又有趣

## 阶段 4：Tile 系统设计

学习：

- autotiling 基础
- 外角
- 内角
- 材质过渡
- 模块化 tile 组合
- 图集组织方式

目标：

- 做出可复用的地板和墙 tile，而不是每次都重画

## 阶段 5：渲染与工具

学习：

- canvas 渲染基础
- 绘制顺序
- 命中检测
- 相机与缩放
- culling
- tile 索引
- 数据格式

目标：

- 从 sprite sheet 正确渲染 tilemap

## 阶段 6：高级打磨

学习：

- 装饰 pass
- 光照 pass
- 动画 tile
- 覆盖层效果
- 性能优化
- 制图工作流

目标：

- 把功能地图打磨成成品地图

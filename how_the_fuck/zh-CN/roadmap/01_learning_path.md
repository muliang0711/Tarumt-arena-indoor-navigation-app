# 学习路线

## 实用路线图

### 第 1 阶段：补基础

重点学习：

- 像素艺术基础
- tileset 规则
- 地图可读性

产出目标：

- 一张很小的手工像素地图图片

### 第 2 阶段：做真正的 tile 渲染器

升级你的应用，让它支持：

- 加载 tile atlas
- 把 tile ID 映射到图集坐标
- 渲染多图层
- 放置装饰物

产出目标：

- 一个通过 sprite sheet 渲染的 JSON tilemap

### 第 3 阶段：把图结构数据和手工地图数据结合起来

图结构数据负责：

- 导航逻辑
- 房间元数据
- 路线计算
- 设施放置规则

手工 tilemap 数据负责：

- 地面外观
- 墙体外观
- 装饰
- 阴影
- 精致度

产出目标：

- 一个既有导航能力、又看起来像手工制作的地图

### 第 4 阶段：提升到可交付质量

加入：

- 更好的标签
- 更多 tile 变体
- 阴影 pass
- 物件分层
- 导入导出流程

产出目标：

- 一个真正像成品的地图查看器，而不是原型

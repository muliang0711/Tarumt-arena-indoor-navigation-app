# Map Engine Localization

## Purpose

`localization/` will own user-position estimation and correction pipelines.

This is where the prototype should eventually centralize dead reckoning, sensor fusion, map matching, and external correction sources such as WiFi fingerprinting.

## Intended Ownership

Examples of future ownership:

- dead reckoning
- sensor fusion
- map matching
- correction pipelines
- confidence and drift handling

## Not For

- page flow
- crowd synchronization
- camera rules

## Current Phase

Scaffold only. No runtime code has been moved here yet.

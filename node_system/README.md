# Node System

This folder stores backend graph output generated by `village_map_editor_next`.

## Output Files

- `nodes.json`: navigation points exported as backend `Node` entities.
- `edges.json`: graph relationships exported as backend `Edge` entities.

## Entity Shape

```json
{
  "node_id": "entrance",
  "floor_id": 1,
  "x": 10,
  "y": 12,
  "type": "destination",
  "name": "Entrance",
  "enabled": true
}
```

```json
{
  "edge_id": "entrance_hall",
  "from_node": "entrance",
  "to_node": "hall",
  "bidirectional": true,
  "weight": 5,
  "distance_m": 5,
  "time_s": 0,
  "accessibility": "standard",
  "enabled": true
}
```

`algorithms/routing.js` and `algorithms/positionDefinition.js` are placeholders for backend logic.
